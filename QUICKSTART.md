# 🚀 Quick Start Guide - Online Exam Proctor System

## Prerequisites Check

- ✅ Python 3.8+ installed
- ⚠️ MongoDB installed and running
- ✅ Dependencies installed (`flask-session`, `pyjwt`, etc.)

## How to Run

1. **Download Mongodb**
In windows:
a. Download MongoDB Community Server: <https://www.mongodb.com/try/download/community>
b. Download Mongodb Compass: <https://www.mongodb.com/try/download/compass>
c. Open Services to check if Mongodb is running by pressing `Win+R` and enter
d. If not then right click it and start it

2. **Start MongoDB**

```bash
Check if MongoDB is running
- Windows: Services panel or 'sc query MongoDB'
- Linux/Mac: 'systemctl status mongod'

Start MongoDB (If Not Running)
- Windows: net start MongoDB  # Windows
- wLinux/Mac: sudo systemctl start mongod  # Linux/Mac
```

3. **Open Terminal in your IDE**

```bash
cd "c:\Users\<USER NAME>\Desktop\ONLINE EXAM PROCTOR SYSTEM"
```

4. **Initialize Project**
Run these commands in terminal
```bash
uv init => for initialize your project(you can remove main.py and readme.md generated by it)
uv venv => to create a virtual environment so that it won't affect your pc
.venv/Scripts/activate => to get into your virtual environment
uv pip install -r requirements.txt => to install every dependency
```

1. **Run Application**

```bash
python app.py
```

5. **Open Browser**

```bash
http://localhost:5000
```
